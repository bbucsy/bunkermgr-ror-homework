# Bunker Manager
This is my Homework project for a Ruby on Rails class. 
The app lets you create bunkers/vaults manage their inventory
and get statistics.

# Technical deteails
## Ruby version
- Ruby: ruby 2.7.1
- Rails: Rails 6.0.3.3
## Database
Mysql (There is a docker-compose file for development)
## Setup
```
bundle install
docker-compose up --build
rails -s
```
